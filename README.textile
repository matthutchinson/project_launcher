h1. Project Launcher

h2. About

A simple, small ruby script to launch commands on a single multi-tabbed OSX Terminal. Project configuration is set in projects.yml, where detection (based on folder precedence) and terminal commands can be setup.  See below for information on installation/usage and setting up. For anything else check the latest commit logs.
